# Landing Page - NextJS
- Criando uma landing page com nextJS.

# Executar o projeto
1 - Clonar em sua máquina.
2 - Instalar todas a dependências com o comando: ``` npm install ```
3 - Executar o projeto usando o comando: ``` npm run dev ```
